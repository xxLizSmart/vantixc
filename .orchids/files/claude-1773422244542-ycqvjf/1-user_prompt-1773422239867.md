Objective: Create a professional service announcement modal for Vantix Pro. The modal should notify users of temporary system instability while emphasizing data security and a clear resolution timeline.

1. Timing & Animation Logic

Delay: The modal must appear exactly 2 seconds after the initial page load.

Animation: Use a "Scale-and-Fade" entrance (from 95% to 100% scale) with a duration of 500ms.

Backdrop: Implement a heavy background blur (backdrop-blur-md) to dim the rest of the site and focus the user on the message.

Container: A centered Glassmorphism card with a deep obsidian background (#080808 at 90% opacity).

Border: A 1px solid border with a soft Electric Cyan glow effect.

Close Button: A minimalist 'X' icon in the top-right corner that allows users to dismiss the modal.

Icon: Include a "Shield" or "Alert" icon at the top in Electric Cyan.

3. Professional Content & Messaging

Headline (Bold): System Maintenance & Security Optimization

Body Text:

"Vantix Pro is currently undergoing critical bug fixes and essential security infrastructure upgrades to ensure a superior trading experience.

Security Note (High Emphasis):

"🔒 Security Note: Your personal data and digital assets remain fully secured and unaffected by these updates."

4. Functional Requirements

One-Time Display: Once the user clicks the "Close" button, store a value in sessionStorage so the modal does not reappear until they start a new browser session.

Responsive: Ensure the modal is perfectly centered and readable on both Desktop and Mobile devices.

Accessibility: Ensure the "Close" button has a high-contrast focus state for keyboard navigation.
